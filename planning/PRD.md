# AI Smart Todo MVP PRD

작성일: 2026-07-18

## 1. 프로젝트 목적

AI Smart Todo MVP는 사용자가 자연어로 입력한 할 일을 AI가 구조화된 todo 초안으로 변환하고, 사용자가 검토 후 저장할 수 있게 하는 단일 사용자 todo 앱이다.

이 MVP의 목적은 기본 todo 관리 흐름과 AI 멀티 에이전트 실행 흐름이 하루 안에 구현 가능한 수준에서 함께 동작함을 검증하는 것이다.

## 2. 문제 정의

일반 todo 앱은 사용자가 작업명, 우선순위, 마감일, 하위 작업을 직접 정리해야 하므로 입력 부담이 크다.

AI를 바로 저장 흐름에 연결하면 잘못된 결과가 실제 todo 데이터로 들어갈 수 있고, 어떤 처리 과정을 거쳤는지 추적하기 어렵다.

따라서 MVP는 AI가 todo 초안을 제안하되, 최종 저장은 사용자가 승인하는 구조로 만든다.

## 3. 대상 사용자

대상 사용자는 개인 작업을 빠르게 정리하려는 단일 사용자다.

사용자는 긴 설명이나 메모 형태로 작업을 입력하고, 앱이 이를 실행 가능한 todo로 정리해 주기를 기대한다.

## 4. MVP 기능

- Todo 생성, 조회, 수정, 완료 처리, 삭제
- 자연어 입력을 구조화된 todo draft로 변환
- draft에 작업명, 설명, 우선순위, 마감일, 하위 작업 제안 포함
- 사용자가 AI draft를 수정하거나 승인해 실제 todo로 저장
- 저장 전 draft와 저장된 todo를 명확히 분리
- AI 처리 요청별 trace id 생성
- 에이전트별 입력, 출력, 오류, 최종 결과 로그 기록
- AI 처리 실패 시 사용자가 직접 todo를 작성할 수 있는 fallback 제공

## 5. 비기능 요구사항

- 단순성: 하루 안에 구현 가능한 최소 흐름만 제공한다.
- 추적 가능성: 자연어 입력부터 draft 생성, 승인, 저장까지 요청 단위로 확인 가능해야 한다.
- 안정성: AI 출력 검증에 실패해도 기존 todo CRUD는 계속 동작해야 한다.
- 테스트 가능성: todo 도메인 로직과 AI 오케스트레이션 로직은 분리한다.
- 데이터 안전성: AI 생성 결과는 사용자 승인 전 실제 todo로 저장하지 않는다.

## 6. Out of Scope

- 다중 사용자 협업
- 로그인, 권한, 역할 관리
- 캘린더, 이메일, 메신저 등 외부 서비스 연동
- 음성 입력
- 모바일 네이티브 앱
- 복잡한 반복 일정과 워크플로 자동화
- 고급 리포팅 또는 분석 대시보드
- 에이전트 마켓플레이스 또는 플러그인 시스템

## 7. 성공 기준

- 사용자가 todo를 생성, 조회, 수정, 완료, 삭제할 수 있다.
- 자연어 입력으로 todo draft가 생성된다.
- 사용자가 draft를 검토, 수정, 승인한 뒤 실제 todo로 저장할 수 있다.
- AI 결과가 승인 전 자동 저장되지 않는다.
- AI 요청의 trace id와 에이전트 실행 로그를 확인할 수 있다.
- AI 처리 실패 또는 출력 검증 실패 시에도 사용자가 수동으로 todo를 작성할 수 있다.
